A minimal and bright hyprland Linux rice.

## Features

eww widget: Running a set of custom scripts.

Swaybg: Dual-background setup with main color and shadow overlay.

Vicinae: Main menu, with default light theme for a clean look.

Firefox: Nightfox theme with light mode enabled.

Kitty: Default Dayfox theme.
icons: Fluent-yellow
